# BTORN ｜ 星屑收藏家的小窝

> 这里是 BTORN 的小宇宙喵~  
> 一个住在云朵里的女孩，每天偷偷收集闪光、糖果回忆和月光碎片。

这是一个用于测试的个人网站，不保证持续可用性或适用于任何特定用途。

在线预览：[https://stinggrey.github.io/btorn-portfolio/](https://stinggrey.github.io/btorn-portfolio/)

## 关于这个小窝

这不是一个普通的页面，而是一扇通向小宇宙的窗户。
推开它，会看到：

- 一颗慢慢打转的小星球，正在悄悄醒过来
- 看板娘从光里软软地飘出来，跟你打招呼
- 一格一格的小梦境，被收进会动的小本本里
- 花园午后的旧电视，画面偶尔晃一下，花瓣就飘起来
- 月亮档案柜里，藏着今天的小心事

## 小窝的角落

```text
小宇宙地图
├─ 看板娘登场       从光里悄悄飘出来
├─ 梦境分类         糖心绘卷 / 一格一格的梦 / 小心心活动 / 我的小宝物
├─ 花园午后         藏在旧电视里的一小段夏天
├─ 月亮档案         一张一张会呼吸的月亮卡片
├─ 小玩具陈列室     糖糖们的小舞台
├─ 闪光宝物柜       每张图都有自己的小位置
├─ 创作日记         一页一页翻开来
├─ 心动小档案       好看的画一张张推到面前
└─ 悄悄话           欢迎打个招呼哦
```

## 怎么进来玩

```powershell
npm install
npm run dev
```

默认会监听整个局域网，手机也可以访问本机显示的地址。

## 打包

```powershell
npm run build
npm run preview
```

仓库只保留页面实际使用的精选资源。本地截图、日志和缓存通过 `.gitignore` 排除。
