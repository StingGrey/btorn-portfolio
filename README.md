# BTORN ｜ 星屑收藏家的小窝

> 这里是 BTORN 的小宇宙喵~
> 一个住在云朵里的女孩，每天偷偷收集闪光、糖果回忆和月光碎片。

## 关于这个小窝

这不是一个普通的页面，而是一扇通向小宇宙的窗户。
推开它，会看到——

- 一颗慢慢打转的小星球，正在悄悄醒过来
- 看板娘从光里软软地飘出来，跟你打招呼
- 一格一格的小梦境，被收进会动的小本本里
- 花园午后的旧电视，画面偶尔晃一下，花瓣就飘起来
- 月亮档案柜里，藏着今天的小心事
- 还有一枚会摇摇晃晃的小挂牌，可以拖着它玩哦

## 小窝的角落

```text
小宇宙地图
├─ 看板娘登场       从光里悄悄飘出来
├─ 梦境分类         糖心绘卷 / 一格一格的梦 / 小心心活动 / 我的小宝物
├─ 花园午后         藏在旧电视里的一小段夏天
├─ 月亮档案         一张一张会呼吸的月亮卡片
├─ 摇晃的小挂件     可以拖着玩的软软挂牌
├─ 小玩具陈列室     糖糖们的小舞台
├─ 会跑的心跳层     无限滚动的小心情
├─ 闪光宝物柜       每张图都有自己的小位置
├─ 会呼吸的字       字也是会跳的小零件呀
├─ 小心思收藏       藏在细节里的小巧思
├─ 创作日记         一页一页翻开来
├─ 心动小档案       好看的画一张张推到面前
└─ 悄悄话           欢迎打个招呼哦
```

## 怎么进来玩

```bash
npm install
npm run dev
```

跑起来之后，会得到一个本地小地址，把它复制到浏览器里就能进来玩啦~
默认会监听整个局域网，手机靠过来也能看哦。

## 把小窝打包带走

```bash
npm run build
npm run preview
```

## 小心事

仓库里只放了页面真正会用到的精选画面。
本地玩耍时产生的原始大图、截图、小日志和缓存都被悄悄藏到 `.gitignore` 里啦，
这样小窝才不会变得沉甸甸的~

要是你也喜欢粉粉的、夜风、还有一点点闪光，
那就把这里当成一扇通向小宇宙的窗吧 ✦
